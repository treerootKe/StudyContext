git指令及其解释：

1.
git config --global(local) user.name "Your Name"
设置全局（局部）用户名

2.git config --global(local) user.email "email@example.com"
设置全局（局部）邮箱地址

3.git init
初始化版本库

4.git add <filename>

5.git commit 

6.git clone <url>

7.$本地仓库、远程仓库相互关联
git remote add origin <url>
git push -u origin <branchName> //-u 将<branchName>设置为上游分支
